# 🏡 WaltHome - Real Estate Landing Page

A modern and responsive real estate landing page built using **HTML**, **CSS**, and **JavaScript**. The project features a clean user interface, smooth interactions, and a mobile-friendly layout designed to showcase premium properties and real estate services.

## 🚀 Live Demo

🔗 https://adelmhmd77.github.io/walthome/

## ✨ Features

- Responsive design for all devices
- Modern and clean UI
- Smooth scrolling and animations
- Interactive navigation
- Property showcase sections
- Testimonials section
- Contact section
- Optimized for performance

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
